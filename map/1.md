## Enunciado
### C. Two TVs
time limit per test
2 seconds
memory limit per test
256 megabytes

Polycarp is a great fan of television.

He wrote down all the TV programs he is interested in for today. His list contains n shows, i-th of them starts at moment li and ends at moment ri.

Polycarp owns two TVs. He can watch two different shows simultaneously with two TVs but he can only watch one show at any given moment on a single TV. If one show ends at the same moment some other show starts then you can't watch them on a single TV.

Polycarp wants to check out all n shows. Are two TVs enough to do so?
Input

The first line contains one integer n (1 ≤ n ≤ 2·105) — the number of shows.

Each of the next n lines contains two integers li and ri (0 ≤ li < ri ≤ 109) — starting and ending time of i-th show.
Output

If Polycarp is able to check out all the shows using only two TVs then print "YES" (without quotes). Otherwise, print "NO" (without quotes).

## Explicación


```cpp
int cantidad_programas;
    cin >> cantidad_programas;

    map<int, int> Ltiempo;

    for (int i = 0; i < cantidad_programas; ++i) {
        int inicio, fin;
        cin >> inicio >> fin;

        Ltiempo[inicio]++;

        Ltiempo[fin + 1]--;
    }
```

Creamos una variable int cantidad_programas para guardar el primer número que nos indica el numero de programas. Luego declaramos una variable map Ltiempo para guardar los valores de inicio y fin de un programa. El Bucle for sirve para poder llenar los elementos del map. Se crea un int incio y otro fin que serviran como clave y valor del map. Ltiempo\[inicio]++ Por cada programa que leemos, sumamos 1 y Ltiempo\[fin + 1]-- el proble indica que si un programa termina en el mismo instante que otro comienza no pueden compartir el televisor. por tanto el televisor se libera recien en el segundo inmediatamente posterior fin + 1 de esto restamos 1 para señalar que el televisor esta disponible nuevamente.

```cpp
 int televisores_actuales = 0;
    bool es_posible = true;
    
    for (auto const& [tiempo, cambio] : Ltiempo) {
 
        televisores_actuales += cambio;

```

Este bloque inicializa un acumulador televisores_actuales en 0 y una bandera es_posible en true, para luego recorrer cronológicamente cada evento del mapa ordenado mediante un bucle. En cada segundo registrado, actualiza el total de televisores sumando los nuevos inicios (+1) o restando las liberaciones (-1); si en algún momento este acumulador supera los 2 televisores disponibles, la bandera cambia a false y el bucle se interrumpe inmediatamente con un break, concluyendo que es imposible ver todos los programas de forma simultánea.

Esta ultima parte del codigo sirve para imprimir la respuesta, si es_posible es true imprime YES, en caso no sea posible inprime NO.
